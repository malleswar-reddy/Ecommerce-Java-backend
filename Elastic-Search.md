



# Elasticsearch, Kibana, and Logstash on Ubuntu (Docker)

## 1. Build and Run Ubuntu Docker Container

```sh
# Build the Docker image
docker build -t java-dev-ubuntu .

# Run the container (based on the ubuntu-ssh image)
docker run -d -p 2222:22 --name ubuntu-container java-dev-ubuntu


2. Connect to the Container via SSH (from Windows)
Use MobaXterm or any SSH client.
Host: 127.0.0.1
Port: 2222
Username: devuser
Password: devpass


3. Prerequisites (Inside the Container)
sudo apt update
sudo apt install -y curl gnupg
curl: Needed to download the GPG key from Elastic
gnupg: Provides gpg for key processing


4. Add Elastic GPG Key and Repository
sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://artifacts.elastic.co/GPG-KEY-elasticsearch | \
  gpg --dearmor | sudo tee /etc/apt/keyrings/elastic.gpg > /dev/null

echo "deb [signed-by=/etc/apt/keyrings/elastic.gpg] https://artifacts.elastic.co/packages/8.x/apt stable main" | \
  sudo tee /etc/apt/sources.list.d/elastic-8.x.list

sudo apt update


5. Install Elasticsearch, Kibana, and Logstash
sudo apt install -y elasticsearch kibana logstash


6. Start Services (No systemd in Docker)
sudo service elasticsearch start
sudo service kibana start
sudo service logstash start


7. Test Elasticsearch
curl -X GET "localhost:9200"
Or with authentication (replace password as needed):
curl -u elastic:<password> https://localhost:9200 --insecure

8. Check Service Status
sudo service elasticsearch status
sudo service kibana status
sudo service logstash status


9. Retrieve Elastic User Password
sudo grep 'password for the elastic user' /var/log/elasticsearch/elasticsearch.log


10. Check Logstash Version
sudo /usr/share/logstash/bin/logstash --version
This format is ready for a `README.md` file.

1)Add Elastic GPG Key and Repository
sudo apt update
sudo apt install -y wget gnupg apt-transport-https
wget -qO - https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo apt-key add -
echo "deb https://artifacts.elastic.co/packages/8.x/apt stable main" | sudo tee /etc/apt/sources.list.d/elastic-8.x.list

2)install elasticsearch
sudo apt update
sudo apt install elasticsearch

3)set file permissions
sudo chown -R devuser:devuser /etc/elasticsearch /usr/share/elasticsearch /var/lib/elasticsearch /var/log/elasticsearch

4)start manually
nohup /usr/share/elasticsearch/bin/elasticsearch > elastic.log 2>&1 &

5)wait and check
curl -k https://localhost:9200 -u elastic:<auto-generated-password>

to reset the password
/usr/share/elasticsearch/bin/elasticsearch-reset-password -u elastic

6)  Install Logstash
sudo apt install logstash

You can check version with:
/usr/share/logstash/bin/logstash --version

🛠️ 4. Configure Logstash
Create a simple config file:
nano logstash.conf

Example content:
input {
  stdin { }
}
output {
  stdout { codec => rubydebug }
}

Run Logstash:
/usr/share/logstash/bin/logstash -f logstash.conf
Type something, and Logstash should process it.

🔧 5. Install Kibana
sudo apt install kibana

If you see error running Kibana like:
no such file or directory, open '/usr/share/kibana/config/kibana.yml'

🛠️ 6. Configure Kibana
Step 6.1: Create config
sudo mkdir -p /usr/share/kibana/config
sudo nano /usr/share/kibana/config/kibana.yml
Paste this:
server.host: "0.0.0.0"
elasticsearch.hosts: ["https://localhost:9200"]
elasticsearch.username: "elastic"
elasticsearch.password: "<your-elastic-password>"
elasticsearch.ssl.verificationMode: "none"


🚀 7. Start Kibana
nohup /usr/share/kibana/bin/kibana --allow-root > kibana.log 2>&1 &
Then:
curl http://localhost:5601

Or open in browser:
http://localhost:5601
